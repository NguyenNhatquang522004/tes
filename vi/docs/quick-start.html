2ï¸âƒ£ Start with Docker Composeâ€‹

  • MongoDB (NGSI-LD storage)

  • Kafka + Zookeeper (streaming)

  • React Frontend

  1. Click "Analytics" in sidebar

    • Hourly traffic volume
    • Accident frequency by location
    • Weather correlation
    • Speed distribution
    • Pattern recognition
  3. Use filters to customize view

  1. Open browser console (F12)

    [WS] Received: {"type": "accident", "data": {...}}

  3. Data should update automatically without page refresh

ğŸ”§ Troubleshootingâ€‹

Services Not Startingâ€‹

# Check Docker resources
docker info

# Restart specific service
docker-compose restart neo4j

# View service logs
docker-compose logs -f backend

Port Conflictsâ€‹

If ports are already in use, edit docker-compose.yml:

# Example: Change Neo4j port
services:
  neo4j:
    ports:
      - "17474:7474"  # Change 7474 to 17474

Database Not Initializedâ€‹

# Stop all services
docker-compose down

# Remove volumes (WARNING: deletes all data)
docker-compose down -v

# Restart
docker-compose up -d

Frontend Not Loadingâ€‹

# Check frontend container
docker-compose logs frontend

# If using local dev server
cd apps/traffic-web-app/frontend
npm install
npm run dev
