Get started in 3 steps (5 minutes):

Option 1: Docker Compose (Recommended)â€‹

# Backend API: http://localhost:8001

# Fuseki: http://localhost:3030

# Start orchestrator
python main.py

# Frontend (separate terminal)
cd apps/traffic-web-app/frontend
npm install
npm run dev
